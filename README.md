PROJECT DESCRIPTION

The Book My Stay App is a console-based Hotel Booking
Management System designed to demonstrate the practical
application of Core Java programming concepts.

The project is developed incrementally through multiple
use cases. Each use case introduces a new concept that
helps developers understand how real-world software
systems are designed and structured.

This implementation focuses on the entry point of a
Java application and demonstrates how execution begins
through the main() method.

This implementation covers Use Case 1:
Application Entry & Welcome Message.

============================================================

USE CASE 1: APPLICATION ENTRY & WELCOME MESSAGE

At this stage, the application demonstrates how a Java
program begins execution.

The program:

- Defines the main application class
- Implements the main() method
- Displays a welcome message
- Shows the application name and version
- Terminates after displaying the message
